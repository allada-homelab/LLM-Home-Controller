We are starting a new repository! The goal of this repository is to create an app that I can use in HomeAssistant that I can use with Assist as a Conversation Agent.

The goal is to make the absolute best experience for llm based smart home control as a conversational agent. As a base, it should be openai api compliant so that I can use my local ai for the inference.

Before we start, I want you to look online for HomeAssistant documentation on how to build custom apps/extensions and specifically ones that you can use with Assist Conversation agent.

Lets init a new python project with uv and python 3.14, a dev container, a .gitignore, and then a research.md on the research you find about how to do this. Spawn subagents to do these in parallel
